退換貨政策 

商品退換貨條件

欲退貨商品，請注意須符合以下條件才能辦理退貨。

(1) 全新商品 : 鑑賞期非試用期，商品經使用將無法退貨，商品收到後請先確認外觀是否為瑕疵，經拆裝使用後怒無法辦理退貨。

(2) 完整包裝 : 商品退貨時必須回復原狀，亦即必須回復至收到商品時的原始狀態 ( 包含商品、配件、贈品、發票、原廠包裝
              ( 紙、盒、袋 ) 、內附文件或資料的完整性等 ) ，若商品被認定為已逾越檢查商品之必要程度，將影響退貨
              權利的行使權或需負擔因損壞產生之費用。

(3) 鑑賞期內 : 取貨後的七天內。 ( 依消保法規定，實體店面購買取貨不適用七天鑑賞期 )

(4) 非商品瑕疵或賣家出貨錯誤等因賣家疏失導致情況下，如消費者有退換貨需求需自行負擔退換運費。 ( 請勿使用到付包裹寄
    送，會直接拒收哦！ )

(5) 為避免接觸性傳染疾病，請購買前先考量好自身毛小孩使用需求，以免造成日後買賣糾紛，如您的商品已經拆封使用，恕無法
    退換貨。

(6) 如當初訂單有達免運資格，在退換部分商品之後訂單金額未達當初免運金額門檻，仍將於退貨退款時扣除原訂單之物流費80元
    。 ( 全退者除外 )

(7) 若原先符合滿額贈活動，後因退貨而未達活動標準時，也須將滿額贈品一併退回。

 

無法退貨情形

如有以下情形者怒無法辦理退貨，若寄回發現商品未有瑕疵或不滿足退貨條件，消費者必須支付運送過程所產生的相關費用。

(1) 超過七天鑑賞期

(2) 非全新商品 : 鑑賞期非適用期，若商品使用後造成無法轉賣，怒不提供退換貨。

(3) 非完整包裝

(4) 活動贈品未完整退回 

(5) 螢幕色差、極小的污點等，在國際驗貨標準皆是屬於可接受的範圍，並不屬於瑕疵 

(6) 商品因個人不當使用拆裝等人為因素造成之故障、損毀、磨損、擦傷、刮傷、髒污、包裝破損不完整或是配件不齊等者。

(7) 接觸型商品 : 鑑賞期非試用期，商品經使用將無法退換貨，商品收到後請先確認外觀是否為瑕疵，經拆裝使用後怒無法辦
    理退貨。屬於個人衛生用品，已拆封使用無法接受退換貨。 ( 例如 : 寵物床墊、寵物衣物、寵物推車、項圈等商品 )

(8) 已拆封商品 : 基於個人衛生疑慮與保障顧客權益，商品經過拆封後無法因不喜歡、等等因素而辦理退換貨，除非拆裝時商
    品內容有瑕疵，否則一旦開封恕不提供退貨。

(9) 毀損、短少商品 : 商品已經經過消費者加工、毀損或者是配件短少等商品。

 

P.S. 若有收到上述無法退貨之商品，將於計算退貨退款時扣除該商品金額。或因最終商品已非完整無法退貨，mao 會將貨品寄
還於您，而此趟寄回給您的運費 ( 貨到付款運費130元起 ) 將由您自行負擔，因此退回商品時請務必多加留意商品的完整性。

 

退款時間說明

◎ 信用卡付款

(1) 取消訂單 : 將於收到您的取消訂單需求後的7-14個工作天為您完成退刷，請留意本期或下期帳單及相關EMAIL通知信件。 
               ( 實際入帳時間要依銀行作業流程 )

(2) 申請退貨 : 將於收到商品確認可辦理退款後的7-14個工作天內為您完成退刷，請留意本期或下期帳單及相關EMAIL通知信件。
               ( 實際入帳時間要依銀行作業流程 )

若經由超商退貨便退貨，整體退貨時間約需14-21個工作天，還請您耐心等候。

 

◎ 超商取貨付款、ATM付款

(1) 申請退貨 : 將於收到商品確認可辦理退款後的7-14個工作天內，將款項退回至您的指定的帳戶內。

若經由超商退貨便退貨，整體退貨時間約需14-21個工作天，還請您耐心等候。

非商品瑕疵或賣家出貨錯誤等因賣家疏失導致情況下，如需使用退貨變將於退款款項扣除75元退貨運費。
